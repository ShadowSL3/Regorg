# Notifu 
**Notifu is Editor Music procedural Made with Love2d**

# Features 
- [x] Default Waves
- [x] Default Note
- [x] GRID
- [ ] Select to drag
- [ ] Custom size Block
- [ ] Options Project
- [ ] Save File

# How to Run 
1. install lua
```
winget install DEVCOM.Lua
```
2. Downlosd & install [LÖVE](https://love2d.org/)
3. Add LÖVE to environment path
4.  Run
```
love .
```
